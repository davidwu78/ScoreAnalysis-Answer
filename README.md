# Coach_AI 

Clone the project 
```
$ git clone https://coachai.cs.nycu.edu.tw:60446/coach-ai/frontend-web.git
$ cd frontend-web
```

Install packages
```
$ yarn install
```

Start
```
$ yarn start
```

Manually add `.env` file to project root!
