# Coach_AI 

Clone the project 
```
$ git clone https://github.com/davidwu78/coachaiimp.git
$ cd frontend-web
```

Install packages
```
$ yarn install
```

Start
```
$ yarn start
```

Manually add `.env` file to project root!
